# Execution Parity Thresholds

The institutional limits for exact bounding without hardcoded magic numbers:

- `Exact Match`: `≤ 1.0`
- `Close Match`: `≤ 3.0`
- `Warning`: `≤ 5.0`
- `Mismatch`: `≤ 10.0`
- `Critical Mismatch`: `> 10.0`

Values bounded structurally in `ParityThresholds::institutional()`.
