# Execution Health Model

The Validation Health metric provides a holistic overview of system integrity.
The state is derived dynamically from:
- Parity Checks
- Determinism Proofs
- Replay Correctness
- Stress Tests
- Benchmarks

Levels:
- `Excellent`: 5/5 components pass
- `Good`: 4/5
- `Normal`: 3/5
- `Weak`: 2/5
- `Critical`: < 2 passes
